خصّص الإعدادات

عدّل هذه القيم لتناسب بياناتك:

التعديلات المطلوبة 
// في Gmail Node — غيّر البريد المستقبل:
"sendTo" → ضع بريدك بدل user@example.com

// في Google Sheets Trigger — اختر جدولك:
"documentId" → اختر ملف Google Sheet الخاص بك
"sheetName" → اختر اسم الورقة

// في Telegram Nodes — ضع Chat ID:
"chatId" → ضع معرّف المحادثة أو المجموعة

// في Jira Node — اختر المشروع:
"project" → اختر مشروعك من القائمة

// في Wait Node — خصّص وقت الانتظار:
"amount": 2, "unit": "hours" → غيّر حسب حاجتك
