من Trello إلى Sheets في 5 خطوات تلقائية

نظام بسيط وفعّال يعمل يومياً في الخلفية — بدون أي تدخل منك.

1

المشغّل الزمني (Schedule Trigger)

N8N يشغّل الـ Workflow تلقائياً كل يوم في الوقت الذي تحدده — يمكن تعديله ليعمل كل ساعة أو كل أسبوع حسب حاجتك.

Schedule Trigger يومي / مخصص
2

قراءة بيانات Trello

3 خطوات متسلسلة: Get Board لجلب اللوحة → Get Lists لجلب جميع القوائم → Get Cards لجلب كل البطاقات (المهام) من كل قائمة.

Trello API Board → Lists → Cards
3

تنظيم البيانات (Map Fields)

يُعيد هيكلة البيانات المستخرجة ويُنظمها في حقول واضحة: اسم اللوحة، اسم القائمة، اسم المهمة، الوصف، تاريخ الاستحقاق، والرابط.

Set Node 6 حقول منظمة
4

إضافة ختم التاريخ (Merge + Today's Date)

يُنشئ تاريخ اليوم بشكل برمجي ويدمجه مع بيانات المهام — لتعرف بالضبط متى تم تسجيل كل لقطة.

Code Node Merge

التسجيل في Google Sheets ✨

جميع المهام تُضاف كصفوف جديدة (Append) في Google Sheets — كل تشغيل يومي يُضيف لقطة جديدة دون حذف البيانات السابقة. سجل تاريخي ينمو يومياً!

Google Sheets Append Mode
البيانات المُسجَّلة

6 حقول تُسجَّل تلقائياً لكل مهمة

كل تشغيل يومي يسحب هذه البيانات من Trello ويُخزنها في Google Sheets.

📅 التاريخ (Date)

تاريخ اليوم الذي تم فيه التسجيل — لتتبع التقدم عبر الزمن.

📑 اسم القائمة (List)

في أي قائمة (To Do، In Progress، Done) المهمة موجودة.

📋 اسم المهمة (Task)

عنوان البطاقة في Trello — ما هي المهمة بالتحديد.

📝 الوصف (Description)

وصف المهمة التفصيلي من البطاقة — تفاصيل إضافية.

⏰ تاريخ الاستحقاق (Due)

الموعد النهائي للمهمة — لتتبع الالتزام بالمواعيد.

🔗 رابط المهمة (URL)

رابط مباشر للبطاقة في Trello — للوصول السريع.

دليل الإعداد خطوة بخطوة

اتبع هذه الخطوات البسيطة لتشغيل المشروع في أقل من 5 دقائق.

1

ربط حساب Trello (Developer API)

Trello يستخدم مفتاح API + Token للاتصال:

أ
اذهب إلى trello.com/app-key ↗ واحصل على API Key
ب
من نفس الصفحة، اضغط على Token لإنشاء رمز الوصول
ج
في N8N → Credentials → New → Trello API ← الصق API Key و Token
2

استورد المشروع في N8N

بعد تحميل ملف JSON:

أ
افتح N8N واضغط على "+ New Workflow"
ب
اضغط (⋮) ← "Import from File" ← اختر الملف المحمّل
3

ربط الحسابات وتخصيص الإعدادات

اضغط على كل Node وأضف بيانات الاتصال:

Trello API

API Key + Token من الخطوة 1

Google Sheets

OAuth2 عبر حساب Google

التعديلات المطلوبة 
// في Get Board Node — غيّر رابط اللوحة:
"value" → ضع رابط لوحة Trello الخاصة بك
مثال: https://trello.com/b/XXXXX/my-board

// في Google Sheets Node — اختر الملف:
"documentId" → اختر ملف Google Sheets من القائمة
"sheetName" → اختر اسم الورقة (Sheet1)

نصيحة: أنشئ Google Sheet جديد بهذه الأعمدة: Date, List Name, Task Name, Task Description, Due Date, url — ثم اختره من Node.

4

فعّل الـ Workflow

آخر خطوة — فعّل الجدولة الزمنية:

أ
اضغط "Test Workflow" أولاً للتأكد من عمل كل شيء
ب
اضغط على "Active" في الأعلى ← سيبدأ العمل تلقائياً كل يوم
ج
(اختياري) عدّل Schedule Trigger لتغيير التوقيت أو التكرار

القالب الحالي مصمم للوحة واحدة. يمكن بسهولة نسخه وتعديل رابط اللوحة لتتبع لوحات متعددة، أو إضافة Loop لجلب عدة لوحات في Workflow واحد.

لا! النظام يستخدم Append Mode — يُضيف صفوف جديدة فقط بدون حذف أي بيانات سابقة. هذا يُنشئ سجل تاريخي كامل لتقدم مشروعك.

بالتأكيد! اضغط على Schedule Trigger Node وغيّر الإعدادات — يمكنك جعله يعمل كل ساعة، مرتين يومياً، أو أسبوعياً. كل شيء قابل للتعديل.
