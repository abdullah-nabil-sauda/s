إنشاء حساب N8N مجاني

دليل الإعداد خطوة بخطوة

1

استورد المشروع في N8N

New Workflow → ⋮ → Import from File → اختر teams-weekly-report.json

2

ربط Microsoft Teams (OAuth2)

في N8N → Credentials → Microsoft Teams OAuth2 → سجّل دخولك وامنح الصلاحيات.

ملاحظة: ستحتاج صلاحيات قراءة رسائل القناة من مسؤول Microsoft 365 في مؤسستك.

3

ربط OpenAI API

Credentials → OpenAI API → الصق مفتاح API الخاص بك من platform.openai.com

4

حدد الفريق والقناة

التعديلات المطلوبة 
// في Fetch Latest Channel Messages Node:
"teamId" → ضع Team ID الخاص بفريقك
"channelId" → ضع Channel ID للقناة المراد تحليلها

// في Send Report to Channel Node:
"teamId" → نفس Team ID
"channelId" → نفس Channel ID (أو قناة مختلفة)
5

فعّل وشغّل! 🚀

اختبر بـ Test Workflow ← تأكد من ظهور التقرير في القناة ← فعّل Active ← سيعمل كل اثنين الساعة 6 صباحاً.

بالتأكيد! الـ System Prompt في الـ LLM Nodes قابل للتعديل بالكامل. يمكنك جعل التقرير رسمياً، مرحاً، مختصراً، أو مفصّلاً — حسب ثقافة فريقك.
