خصّص إعدادات Config Node

افتح Config Node وعدّل القيم التالية:

Config Node — التعديلات المطلوبة 
// معرّف مجلد الصور الجديدة (من رابط Google Drive):
"inputFolderId" → REPLACE_WITH_INPUT_FOLDER_ID

// معرّف مجلد الصور المعالجة:
"processedFolderId" → REPLACE_WITH_PROCESSED_FOLDER_ID

// بريد المستلم للتقارير:
"emailTo" → user@example.com

// عنوان البريد (اختياري):
"emailSubjectPrefix" → Construction Photo Report

// معرّف جدول Google Sheets:
"logSpreadsheetId" → REPLACE_WITH_SPREADSHEET_ID

// اسم الورقة:
"logSheetName" → Sheet1

// لغة التقرير:
"reportLanguage" → Arabic (أو English)
