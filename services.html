إجابات على أكثر الأسئلة تكراراً

يعتمد السعر على مدى تعقيد العملية وعدد التطبيقات المطلوب ربطها. بشكل عام، Workflow بسيط (مثل ربط نموذج بـ Google Sheets + إشعار بريدي) يبدأ من 1,000 ريال. الأنظمة المعقدة التي تتضمن ذكاء اصطناعي قد تصل إلى 5,000+ ريال. أقدم دائماً عرض سعر دقيق بعد فهم احتياجاتك.
نعم بالتأكيد! كل باقة تتضمن فترة دعم مجانية (من 3 أيام إلى شهر كامل حسب الباقة). بعد انتهاء الفترة المجانية، أقدم عقود صيانة شهرية أو سنوية بأسعار تنافسية تشمل التحديثات والإصلاحات والتحسينات المستمرة.
N8N يدعم أكثر من 400 تطبيق بشكل مدمج (Google, Slack, WhatsApp, Shopify, سلة, HubSpot...). وأي تطبيق آخر يوفر API يمكن ربطه بسهولة. حتى لو كان لديك نظام داخلي خاص، يمكنني بناء Custom Node أو استخدام Webhooks للربط. أخبرني بالنظام وسأؤكد لك الإمكانية مجاناً.
يعتمد على حجم المشروع. تدقيق Workflow قائم: 3-5 أيام. بناء Workflow جديد: 1-2 أسبوع. نظام متكامل: 2-4 أسابيع. أحرص دائماً على تحديد جدول زمني واضح في بداية المشروع مع تحديثات مستمرة لك.
الأمان أولوية قصوى. N8N يمكن تشغيله على سيرفرك الخاص (Self-hosted)، مما يعني أن بياناتك لا تغادر أبداً بيئتك. أستخدم دائماً HTTPS للاتصالات، وAPI Keys مشفرة، وأطبق مبدأ الصلاحيات الأقل (Least Privilege). أيضاً أوقّع اتفاقية سرية (NDA) مع كل عميل.
أقبل الدفع عبر التحويل البنكي المحلي (جميع البنوك السعودية)، PayPal، أو Wise للعملاء الدوليين. يتم الدفع عادةً على مرحلتين: 50% مقدماً لبدء العمل، و50% عند التسليم والاعتماد النهائي. للمشاريع الكبيرة يمكن الاتفاق على دفعات مرحلية.
